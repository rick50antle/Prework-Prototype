# Module 4 reference filings

Frozen copies of the financial statements used in Module 4. The tutorial
verifies specific page numbers and dollar figures against these PDFs, so
the files in this directory are the canonical source of truth — do not
swap them for a newer year without also updating Module 4's verified
answers and screenshots.

## Refreshing to a newer fiscal year

When the team wants to point Module 4 at the next year's filings:

1. Download the new PDFs from the source URLs above (which typically
   point to whichever filing is current — i.e., the URLs themselves
   will rotate, but the *content* at those URLs changes year over year)
2. Save them here with a new filename (e.g., `cvs-health-FY2026-10K.pdf`)
3. Re-screenshot the BS / IS / SoCE / CFS pages for the in-tutorial
   "see original" overlays
4. Re-transcribe the statement data into the Module 4 tutorial source
5. Update the verified page numbers and dollar figures in the tutorial
6. Update this README
